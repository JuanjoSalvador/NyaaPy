Unofficial Python module for Nyaa.si (WebScraping) and Nyaa.pantsu.cat (API wrapper)

Supports Python 3+

* [Installation](#installation)
* [Docs](https://github.com/JuanjoSalvador/NyaaPy/wiki)
* [License](#license)

## Installation

Install it using pip.

    pip install nyaapy
